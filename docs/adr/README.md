# ADR Authority

The Frothy ADR series begins at `100`.

Active Frothy decisions:

- `100-*.md` and above

Inherited Froth decision history:

- retained substrate-reference ADRs under `001-060`
- archived Froth ADRs under `docs/archive/adr/`

Authority rules:

- Frothy `100`-series ADRs are normative for Frothy architecture and workflow
  decisions.
- Retained Froth ADRs in `docs/adr/` remain available for substrate reference.
- Archived Froth ADRs remain historical reference only.
- A Froth ADR is not active Frothy policy unless a Frothy ADR explicitly adopts
  it or constrains itself around it.

Use `000-template.md` for new Frothy ADRs.
When citing a Frothy ADR in prose, prefer the form `Frothy ADR-100`.
