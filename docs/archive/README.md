# Archive

`docs/archive/` holds historical Froth-era material and superseded Frothy
notes that still matter as reference evidence.

Use this tree for:

- substrate background when Frothy intentionally reuses inherited behavior
- historical proof transcripts and workshop evidence
- retired design notes that still explain why an old path exists

Do not treat archive contents as authoritative for:

- current Frothy language semantics
- the live roadmap or milestone order
- the maintained product, install, or support surface

Subdirectories:

- `adr/`: archived Froth-era ADRs and retired decisions
- `concepts/`: historical design notes and proposals
- `proofs/`: archived proof transcripts and related evidence
- `spec/`: superseded language/spec material kept for reference only
